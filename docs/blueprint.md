# **App Name**: PocketFlow

## Core Features:

- Intuitive Dashboard: Displays a clear financial overview, current budget progress, and spending habits at a glance.
- Expense & Income Tracking: Facilitates easy and quick entry of all expenses and income, allowing for custom categorization.
- Budget & Goal Setting: Allows users to define spending goals and create monthly budgets for various categories, with progress monitoring.
- Visual Spending Analysis: Generates interactive charts and graphs to visually analyze spending patterns and trends over time.
- AI-powered Transaction Categorization: Uses an AI tool to automatically suggest and learn custom categories for new transactions based on past entries.
- Secure Data Persistence: Securely stores all user financial data in a robust and scalable database for retrieval and analysis.

## Style Guidelines:

- Primary color: A vibrant, fresh green (#29A35C) to signify growth and prosperity, standing out on a light background.
- Background color: A clean, nearly white with a very subtle green tint (#F9FBF9), creating a sense of clarity and spaciousness.
- Accent color: A lively lime-yellow (#E1ED5E) to draw attention to interactive elements and highlights, complementing the primary green.
- Body and headline font: 'Inter' (sans-serif) for its modern, highly legible, and objective aesthetic, suitable for financial data.
- Utilize minimalist line icons for intuitive navigation and data representation, maintaining a clean and professional feel.
- Emphasize a clean and spacious layout with ample whitespace to ensure readability and an intuitive user experience for financial management.
- Incorporate subtle and smooth transitions for chart updates, menu navigation, and expense entry feedback to enhance user engagement without distraction.
